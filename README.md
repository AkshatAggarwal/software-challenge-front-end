### Tasks

##### #1 Sorting Scans:
Create UI and logic to sort the scans by:

- Name
- Username
- Elevation

##### #2 Edit Scan:
Create UI and logic for editing a scan's name and associated user.

##### #3 Add Scan:
Create UI and logic to create new scans

### Build and run:

Install:
#### `npm|yarn install`

Run in development mode:
#### `npm|yarn start `
